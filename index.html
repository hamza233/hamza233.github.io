Clima

The app gives weather updates based on user's location. User can also see the weather of any other city by entering name.

The Core Location module was used to handle user's location, third-party API was used to get the weather data and networking was carried by the Alamofire cocoapod

Hacker News

The Hacker News app shows the front page of https://news.ycombinator.com, the link of article opens up in a web view when user clicks on any of the list items

The app is designed using SwiftUI, the data is taken using https://hn.algolia.com/api/v1/search?tags=front_page api, custom webview was built using the webkit module
